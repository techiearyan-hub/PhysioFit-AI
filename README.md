PhysioFit AI 

PhysioFit AI is a web application that acts as your **personal fitness trainer and dietitian**, powered by AI.  
It generates personalized weekly diet plans based on user input (name, age, height, weight, ailments, preferences, and health goals) and displays them in a structured dashboard.

---

 Features
- Clean UI.
- Input form for personal details and health goals.
- AI-powered **weekly diet plan generation** via OpenRouter API.
- Dashboard view with both raw text and structured daily cards.
- Progress tracking section for future enhancements.

---

 Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **AI API:** OpenRouter (GPT‑4o mini or other models)
- **Styling:** CSS (custom dark theme with orange accents)

---

 Setup
1. Clone the repository:
2. Create a .env file and add API key.
3. cd server > npm start
4. cd client > npm run dev
